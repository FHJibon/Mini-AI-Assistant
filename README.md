# Mini AI Assistant

## ⚡ Quick Start

### 1. Set Environment Variables
- Create a `.env` file in `app/` floder and define the required variables:
```env
PINECONE_API_KEY=
PINECONE_INDEX_NAME=
OPENAI_API_KEY=
OPENAI_MODEL=
```

### 2. Run the Application
- Start the application:
```
docker compose up --build
```

###  API Documentation: **http://localhost:9000/docs**
